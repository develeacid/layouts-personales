Stack tecnologico

Herramientas elegidas por su simplicidad, potencia y cero dependencias de build

Tailwind CSS

Framework utility-first que permite construir cualquier diseño directamente en el HTML. Sin CSS custom, sin naming conventions, maximo control visual.

Alpine.js

Framework reactivo minimal que agrega interactividad directamente en el markup. Ideal para componentes que necesitan estado sin la complejidad de React o Vue.

HTML5 Semantico

Cada componente es un archivo HTML standalone. Sin bundlers, sin transpilers, sin build steps. Abre cualquier archivo en el navegador y funciona.
