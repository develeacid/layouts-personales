# LayoutOS

Sistema de diseño educativo — 107 componentes + 66 páginas construidos con **Tailwind CSS v4** y **Alpine.js 3**.

## Quick start

```bash
npm install
npm run dev
```

## Docs

- `docs/BRANDING.md` — Tokens de diseño, paleta, tipografía, espaciado
- `docs/ROADMAP.md` — Ruta de aprendizaje y orden de construcción
- `docs/SETUP.md` — Configuración técnica y convenciones
