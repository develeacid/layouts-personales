# N8 Charts Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Crear 10 componentes educativos de gráficas (C108–C117) usando Apache ECharts + Alpine.js, con tablas editables, descarga standalone, y sistema de CDN genérico.

**Architecture:** Cada componente es un HTML self-contained con ECharts via CDN. Alpine.js gestiona los datos reactivos (`x-data`) que alimentan las gráficas. El patrón base incluye deep watch, resize handler, y cleanup. El sistema de descarga se amplía para extraer CDNs externos automáticamente.

**Tech Stack:** Apache ECharts 5 (CDN), echarts-stat (CDN, solo C115), Alpine.js 3 (CDN), Tailwind CSS v4

### Task 1: Ampliar sistema de descarga — soporte CDNs externos

**Files:**
- Modify: `src/js/spa-router.js`
- Modify: `src/js/code-download.js`

**Step 1: Modificar parseBody en spa-router.js para extraer CDNs**

En `spa-router.js`, función `parseBody`, agregar extracción de scripts externos antes del `return`:

```js
// Después de la extracción de styles, antes del return:

// Extraer CDNs externos (excepto Alpine)
const cdns = [];
doc.querySelectorAll('script[src]').forEach(s => {
  if (!s.src.includes('alpinejs')) {
    cdns.push(s.getAttribute('src'));
  }
});

return { html: body.innerHTML, scripts, styles, cdns };
```

**IMPORTANTE:** Usar `s.getAttribute('src')` (no `s.src`) para obtener la URL tal como está en el HTML, no la URL resuelta por el browser.

**Step 2: Pasar cdns a injectDownloadButtons**

En `spa-router.js`, línea del `injectDownloadButtons`, cambiar:

```js
// Antes:
injectDownloadButtons(main, path, text, parsed.scripts, parsed.styles);

// Después:
injectDownloadButtons(main, path, text, parsed.scripts, parsed.styles, parsed.cdns);
```

**Step 3: Modificar code-download.js para recibir y usar CDNs**

Cambio 1 — Ampliar regex para aceptar N1-N8:

```js
// Después:
```

Cambio 3 — Guardar cdns junto a scripts/styles:

```js

```

Cambio 4 — Pasar cdns a buildVariantFile:


```js

// Después:
```

Y en el template HTML generado, antes de los scripts inline:

```js
// Antes:
  <style>[x-cloak] { display: none !important; }</style>

  <script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>
```

```html
    function chartBase() {
      return {
        chart: null,
        _resizeHandler: null,
```
