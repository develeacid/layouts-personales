Rich Text Toolbar

Barra de formato de texto enriquecido con editor visual y Markdown.

Variante 1 — Toolbar completa

Selecciona texto y usa la barra de herramientas para darle formato. Prueba con negrita, cursiva y subrayado.

Variante 2 — Toolbar compacta (bubble)

Selecciona cualquier texto de este editor para ver aparecer la barra de formato flotante. Prueba seleccionando esta frase para aplicar negrita, cursiva o insertar un enlace.

Este editor usa una toolbar tipo "bubble" que aparece contextualmente sobre el texto seleccionado, similar a editores como Medium o Notion.
