Preguntas frecuentes

LayoutOS es un sistema de diseño educativo con 107 componentes y 66 páginas, construido con Tailwind CSS v4 y Alpine.js 3, organizado en 7 niveles de dificultad progresiva.

Conocimiento básico de HTML y CSS es suficiente para empezar. Los primeros niveles son completamente estáticos y cada nivel introduce conceptos nuevos gradualmente.

Sí, todo el código es open source. Puedes copiar, modificar y usar los componentes en proyectos personales y comerciales sin restricciones.

Está planificado como mejora futura. La paleta Zinc como base facilita la implementación de dark mode.

Aceptamos PRs en GitHub. Puedes agregar nuevos componentes, mejorar los existentes o reportar bugs a través de Issues.
