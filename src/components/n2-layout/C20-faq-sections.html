Preguntas frecuentes

¿Qué es LayoutOS?

LayoutOS es un sistema de diseño educativo que incluye 107 componentes y 66 páginas construidos con Tailwind CSS y Alpine.js, organizados por niveles de dificultad progresiva.

¿Necesito experiencia previa?

Conocimiento básico de HTML y CSS es suficiente. El nivel 1 empieza con componentes estáticos simples y cada nivel introduce conceptos nuevos gradualmente.

¿Puedo usar los componentes en mis proyectos?

Sí, todos los componentes son open source. Puedes copiar, modificar y usar el código en proyectos personales y comerciales sin restricciones.

¿Qué tecnologías se usan?

Tailwind CSS v4 para estilos y Alpine.js 3 para interactividad. No se requiere bundler ni framework adicional.

¿Se agregará dark mode?

Está planificado como mejora futura. La base actual con Zinc facilita la implementación de dark mode cuando se agregue.

¿Con qué frecuencia se actualiza?

Se agregan componentes nuevos cada semana y se publican releases mensuales.

Todos los componentes son mobile-first y responsive por defecto.
