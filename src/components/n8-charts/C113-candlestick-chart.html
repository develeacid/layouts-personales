C113 · Nivel 8

Candlestick

Nivel 2 · Multiatributo — cada punto contiene 4 valores (Open, High, Low, Close) para análisis financiero.

1 · OHLC básico

Análisis de mercado financiero / crypto — velas japonesas con apertura, máximo, mínimo y cierre diarios.

Estructura de datos

data: [
  { date:'2024-01-02', open:2320.26, close:2302.60, low:2287.56, high:2362.94 },
  { date:'2024-01-03', open:2300.00, close:2291.30, low:2288.26, high:2308.38 },
  ...
]
Date Open High Low Close

2 · Con volumen

Velas japonesas con gráfica de barras de volumen sincronizada debajo, con zoom compartido.

Estructura de datos

data: [
  { date:'2024-01-02', open:2320.26, close:2302.60, low:2287.56, high:2362.94, volume:3842 },
  { date:'2024-01-03', open:2300.00, close:2291.30, low:2288.26, high:2308.38, volume:2156 },
  ...
]
Date Open High Low Close Volume

3 · Con media móvil (SMA 5)

Velas japonesas con línea de media móvil simple de 5 periodos superpuesta para detectar tendencias.

Estructura de datos

data: [
  { date:'2024-01-02', open:2320.26, close:2302.60, low:2287.56, high:2362.94 },
  ...
]
// SMA(5) = promedio de close de los últimos 5 días
Date Open High Low Close

4 · Con zoom

Velas japonesas con slider de zoom y zoom por scroll para explorar rangos de fechas.

Estructura de datos

data: [
  { date:'2024-01-02', open:2320.26, close:2302.60, low:2287.56, high:2362.94 },
  ...  // 20 días de trading con dataZoom slider + inside
]
