C111 · Nivel 8

Gauge

Nivel 2 · Multiatributo — KPIs con umbrales, rangos de rendimiento y múltiples métricas.

1 · Gauge simple

Arco semicircular mostrando un valor porcentual de rendimiento del sistema.

Estructura de datos

data: { value: 72, min: 0, max: 100 }
Rango: 0 – 100

2 · Multi-anillo

Tres gauges concéntricos mostrando CPU, RAM y Disco simultáneamente.

Estructura de datos

data: [
  { name: 'CPU', value: 68 },
  { name: 'RAM', value: 85 },
  { name: 'Disco', value: 42 }
]
Recurso Valor (%)

3 · Con rangos de color

Gauge con umbrales de color que indican el estado del rendimiento: bajo, medio y alto.

Estructura de datos

data: { value: 72 }

// Rangos del eje:
//   0–40  → Rojo   (#ef4444) — Rendimiento bajo
//  40–70  → Amarillo (#f59e0b) — Rendimiento medio
//  70–100 → Verde  (#10b981) — Rendimiento alto
0–40: Bajo 40–70: Medio 70–100: Alto

4 · Dashboard

Cuatro gauges en grid mostrando KPIs clave del negocio con distintas escalas y unidades.

Estructura de datos

data: [
  { name: 'Conversión', value: 3.6, max: 10, unit: '%' },
  { name: 'NPS', value: 72, max: 100, unit: '' },
  { name: 'Uptime', value: 99.9, max: 100, unit: '%' },
  { name: 'Latencia', value: 45, max: 200, unit: 'ms' }
]
Métrica Valor Máximo Unidad
